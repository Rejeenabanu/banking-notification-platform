# Cloud-Native Banking Notification Platform

A production-grade banking notification platform built using:

- Java 21
- Spring Boot
- PostgreSQL
- Redis Streams
- AWS
- Docker
- Terraform
- GitHub Actions
- OpenTelemetry
- Grafana
